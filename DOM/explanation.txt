BOM
- in full is window browser object, it is the element that makes it possible for your javascript 
code to communicate with the browser.

1) window
    The window object contains all the properties required to represent the window of the browser
     example
        size of the window 
        history of previously visited web page
    the exact implementation of the BOM depends on the browser and the version of the browser.
    - important objects of the BOM
         a) history
         b) Navigator
         c) Location
